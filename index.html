FAQ

Preguntas frecuentes

¿Velo es completamente gratuito? ▾
Sí, el plan gratuito incluye todas las funciones esenciales sin límite de tiempo. El plan Pro añade funciones avanzadas para comunidades más grandes.
¿Cómo funciona el sistema de tickets? ▾
El sistema de tickets permite crear categorías personalizadas con menús desplegables. Cada ticket abre un canal privado con el equipo de soporte. Puedes configurar roles, transcripciones automáticas y flujos de cierre con valoraciones.
¿Qué plataformas soportan las notificaciones? ▾
Actualmente Velo soporta YouTube, Twitch, Kick, TikTok e Instagram. Solo necesitas introducir el nombre de usuario y el bot verificará el perfil automáticamente.
¿Cómo configuro el idioma del bot? ▾
Usa el comando !setidioma-velo es para español o !setidioma-velo en para inglés. El cambio se aplica inmediatamente a todos los mensajes del bot en tu servidor.
¿Puedo personalizar los mensajes de bienvenida? ▾
Sí, con /bienvenida-setup puedes configurar el canal, el mensaje con variables dinámicas como nombre del usuario, número de miembros, mención, etc. También puedes personalizar el color del embed y añadir imagen de fondo.
¿Dónde puedo obtener soporte? ▾
Visita nuestra página de soporte o únete al servidor de Discord de soporte para obtener ayuda directa del equipo.
